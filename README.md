# Boas vindas ao repositório do projeto Lista de Tarefas!

O projeto consiste em uma lista de tarefas. O usuário poderá adicionar, remover, marcar como feita e trocar a ordem das tarefas.

## Tecnologias Usadas

- HTML
- CSS
- DOM
-Javascript

## Habilidades

- Manipular CSS
- Manipular Javascript
